Innovation Playground

賈伯斯曾說：「創新即借用與連結。」這是屬於我的技術萌芽基地。記錄我從模仿中磨練技能，進而嘗試將各項技術連結、內化，最終轉化為創意的成長足跡。

《星期天的我們家哦》

大二「遊戲程式設計」課程期末專案。以《蠟筆小新》的角色設定為核心，開發的 ActionScript 互動遊戲。劇情完善並設計關卡與引導動畫，深入探索如何運用極簡圖示實現直覺化導引提示。

ActionScript Flash Legacy Visual Guide
《被迫營業的機器人蟹老闆》

大二「電腦圖學」期末專案。使用 C 語言與 OpenGL 開發，並結合 Maya 主角建模。專案核心在於透過程式精確控制模型色彩、光線與照明參數。作品在期末展示中榮獲全班票選第一名，並在 YouTube 收穫萬次點閱。

C & OpenGL Rendering Popular Project
《虛擬甜點店網站實作》

大二「Web程式設計」作品。結合 HTML、CSS 與 PHP 構建品牌網站。從線框圖規劃頁面佈局，到實作購物車、留言板等動態功能，並深入調教響應式設計（RWD）與滑鼠滑過時的視覺回饋。

HTML / CSS / PHP Responsive UI Planning
《莉露姆做蛋糕》

大三「互動技術概論」期末專案。使用 Java 與 OpenGL 打造結合 2D 顏色混合與 3D 裝飾系統。透過實作攝影機視角切換與打光技術，我初次體會到 3D 空間帶來的沉浸感，也奠定了我對使用者體驗的重視。

Java & OpenGL 3D Interaction
《美冴的一天》
